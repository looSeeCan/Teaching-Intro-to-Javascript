Variables, Syntax, Data types

Mathematics - a quantity which during a calculation is assumed to vary or be capable of varying in value.

Computing - a data item that may take on more than one value during the runtime of a program.

Syntax - Syntax is the set of rules that define what the various combinations of symbols and words mean. This tells the computer how to read the code.

Data Types: String, Boolean, Number, Null, Undefined, Object, Symbol
